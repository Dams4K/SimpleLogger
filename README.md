# Simple Logger

Does nothing except adding 4 static method : Log.debug, Log.info, Log.warn, Log.error

## Installation

### HTTPS
```bash
git submodule add https://github.com/Dams4K/SimpleLogger.git addons/logger
```

### SSH
```bash
git submodule add git@github.com:Dams4K/SimpleLogger.git addons/logger
```
